#  Cronograma Study — AI-Based Interactive Study Planner

Cronograma Studio is an interactive study planning tool designed to make learning more engaging, structured, and motivating.

This project was created as part of my personal learning journey, based on a simple idea: I learn better through interaction, so I decided to build a system that reflects that.

##  Core Concept

The platform combines structured study planning with gamification elements, transforming the learning process into a more dynamic and rewarding experience.

Instead of a traditional static planner, this project introduces an interactive approach where users can actively track their progress, complete tasks, and stay motivated.

##  Key Features

###  Interactive Study Schedule

* Users can create daily plans by listing tasks and goals
* Each task can be marked as completed
* Designed to simulate real day to day study organization

###  Gamification System

* Each completed task rewards the user with XP points
* Progress is tracked based on completed goals
* Encourages consistency and motivation

###  Learning Levels

* Beginner
* Intermediate
* Advanced

Each level is structured to guide users through different stages of learning.

###  Learning Modules

* Python
* Solidity
* Other technology topics

Each module includes structured explanations and progression paths.

###  Mini Games for Learning

* Interactive quizzes and mini games
* Aligned with the chosen study topic
* Reinforces knowledge through practice

### 🌐 Bilingual Support

* Available in English and Portuguese
